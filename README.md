# Retail Business Intelligence Dashboard

## Overview
This project analyzes retail transaction data to uncover key business insights using SQL, Python, and Tableau.

## Key Features
- Revenue trend analysis over time
- Customer segmentation (High, Medium, Low value)
- Top 10 revenue-generating products

## Tools Used
- SQL
- Python (Pandas)
- Tableau

## Key Insights
- Revenue shows consistent growth over time
- Medium-value customers contribute the highest overall revenue
- A small group of high-value customers generates strong individual spending
- Top-performing products drive a large share of revenue

## Project Structure
- data/ → cleaned datasets
- notebooks/ → analysis scripts
- dashboard/ → Tableau dashboard
